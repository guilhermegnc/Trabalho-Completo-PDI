Trabalho completo feito em Python para reconhecimento de padrões usando texturas e também manipulação da imagem usando quantização e equalização

Lembrando que foi utilizado a versão 3.8.5 do Python
==============================================
Foi colocado para instalar as dependências o arquivo requirements.txt com todas as bibliotecas instaladas no computador, a grande maioria não foi utilizada, assim talvez seja mais útil instalar apenas as que são utilizadas no código e que são apresentadas na documentação, porém foi dada essa opção de automatizar ja utilizando o requirements.txt
